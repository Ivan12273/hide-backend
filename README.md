# HIDE backend

HIDE es un sistema de software con el propósito de ayudar a las personas involucradas en el proceso de ventas de mostrador al registro de las ventas, registro de clientes, visualizacón de las existencias de los productos y el manejo de las ordenes. El objetivo de HIDE es ayudar a las personas que colaboran en microempresas a gestionar la información de las ventas de manera efectiva y sencilla.
